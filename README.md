# 🐍 python-project-vault

A growing collection of Python projects — from small scripts to real-world tools.
Each project lives in its own folder with a README, runnable code, and clear explanations.

---

## 📁 Projects

| Project | Description | Difficulty |
|---------|-------------|------------|
| [ai-resume-builder](projects/ai-resume-builder/) | Generate a clean PDF resume from a JSON file | 🟡 Intermediate |

> More projects coming soon — voice assistant, price tracker, PowerPoint generator, and more.

---

## 🛠 Requirements

- Python 3.10+
- Each project has its own `requirements.txt`

---

## 🚀 Getting Started

```bash
git clone https://github.com/joshivignesh/python-project-vault.git
cd python-project-vault

# Navigate into any project
cd projects/ai-resume-builder

# Install dependencies
pip install -r requirements.txt

# Run it
python main.py
```
